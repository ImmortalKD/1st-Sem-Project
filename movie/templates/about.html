Online Movie Ticket Booking System is a website that provides the customers a facility to book tickets for a movie online. Customer needs to register at the site to book tickets to the movie. After selecting the show, the user is presented a seating layout so that he/she can select seats of his/her choice. Thereafter he/she is redirected to the payment gateway for completing the transaction. User can update his/her profile and view his/her booking history.
